# Madam Nails Website

A beautiful, modern one-page website for Madam Nails beauty salon in Pretoria.

## 📁 File Structure

```
madam nails/
├── index.html          # Main HTML file
├── styles.css          # All styling
├── script.js           # JavaScript functionality
├── images/             # Image folder (add your images here)
│   ├── hero-bg.jpg     # Background image for hero section
│   ├── gallery-1.jpg   # Gallery images
│   ├── gallery-2.jpg
│   ├── gallery-3.jpg
│   ├── gallery-4.jpg
│   ├── gallery-5.jpg
│   └── gallery-6.jpg
└── README.md           # This file
```

## 🖼️ Images Needed

Please add the following images to the `images` folder:

### 1. Hero Background (`hero-bg.jpg`)
- A high-quality background image for the main hero section
- Recommended size: 1920x1080px
- Should be professional and represent your brand

### 2. Gallery Images (`gallery-1.jpg` through `gallery-6.jpg`)
- 6 images showcasing your nail work and beauty treatments
- Recommended size: 800x800px (square format works best)
- Showcase different services: manicures, pedicures, nail art, etc.

## 🎨 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Smooth Scrolling**: Navigate smoothly between sections
- **Modern Animations**: Elegant fade-in effects and hover animations
- **Contact Form**: Integrated booking form for appointments
- **Social Media Links**: Direct links to Facebook, Instagram, and WhatsApp
- **Professional Layout**: Clean, elegant design with gold accents

## 🚀 How to Use

1. **Add Your Images**:
   - Place your images in the `images` folder
   - Name them according to the structure above
   - Or update the image paths in `index.html` to match your file names

2. **Open the Website**:
   - Double-click `index.html` to open in your browser
   - Or use a local server for better performance

3. **Customize** (Optional):
   - Update colors in `styles.css` by changing the CSS variables at the top
   - Modify text content in `index.html`
   - Add more services or gallery items as needed
